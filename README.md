# โครงการธรรมาภิบาลข้อมูล (Data Governance) - สทร.
**สถาบันวิจัยและพัฒนาเทคโนโลยีระบบราง (องค์การมหาชน) (สทร.)**
*Railway Technology Research and Development Institute (Public Organization) - RTRDI*

---

ระบบเว็บแดชบอร์ดนำเสนอความคืบหน้าและการจัดทำกรอบแนวทาง แผนดำเนินงาน และการวิเคราะห์ช่องว่าง (Gap Analysis) สำหรับการขับเคลื่อนภารกิจธรรมาภิบาลข้อมูลสถาบัน เพื่อให้สอดรับกับนโยบายดิจิทัลและมาตรฐานความปลอดภัยของรัฐบาล

---

## 🌟 คุณสมบัติเด่น (Key Features)

1. **Interactive Navigation:** แสดงผลแยกหมวดหมู่ตามแท็บย่อย (บทสรุปผู้บริหาร, กรอบอ้างอิงหลัก, วิเคราะห์ช่องว่าง, แผนงาน 6 เดือน และประเด็นเสนออนุมัติ) 
2. **Visual Gantt Chart:** แผนภาพเส้นเวลาแสดงกลุ่มงานย่อย 4 ระยะหลัก ตลอด 6 เดือน เพื่อเพิ่มความชัดเจนในการนำเสนอ
3. **Interactive Gap Tracker:** แผงควบคุมจำลองและติดตามความคืบหน้าของโครงการประเมินความพร้อมแบบเรียลไทม์
4. **Official Print-to-PDF:** จัดทำระบบควบคุม CSS Print Stylesheet อย่างละเอียด สามารถพิมพ์หรือแปลงเป็นเอกสารรูปเล่ม PDF ระดับทางการได้ทันที (ซ่อนปุ่มแถบเมนูนำเสนอและแสดงหน้าปกสถาบันพร้อมตารางเซ็นอนุมัติ)

## 🚀 วิธีการใช้งาน (Getting Started)

### 1. การเข้าดูเว็บแดชบอร์ด
คุณสามารถเข้าชมเว็บแดชบอร์ดนำเสนอได้ทันทีโดยไม่ต้องติดตั้งซอฟต์แวร์เซิร์ฟเวอร์เพิ่มเติม:
*   ดับเบิ้ลคลิกที่ไฟล์ **`index.html`** ในคอมพิวเตอร์ของคุณเพื่อเปิดผ่านเว็บเบราว์เซอร์ทั่วไป (เช่น Google Chrome, Microsoft Edge, Safari)

### 2. การแปลงรายงานเป็นไฟล์ PDF หรือ พิมพ์เอกสาร
เพื่อจัดทำเป็นเอกสารสำหรับนำเสนอระดับบอร์ดบริหาร สทร. ในรูปเล่มรายงานทางการ:
1.  คลิกที่ปุ่ม **"พิมพ์รายงาน / บันทึก PDF"** ที่อยู่บริเวณมุมขวาบนของหน้าเว็บ (หรือกดปุ่มคีย์ลัด `Ctrl + P` บนระบบ Windows หรือ `Cmd + P` บนระบบ macOS)
2.  ที่เมนูการพิมพ์ของเบราว์เซอร์ ให้เลือกปลายทาง (Destination) เป็น **"บันทึกเป็น PDF" (Save as PDF)**
3.  ตรวจสอบการตั้งค่า:
    *   **เค้าโครง (Layout):** แนวตั้ง (Portrait)
    *   **ขนาดกระดาษ (Paper size):** A4
    *   **ตัวเลือกเพิ่มเติม (Options):** ติ๊กถูกที่ช่อง **"กราฟิกพื้นหลัง" (Background graphics)** เพื่อให้แถบสีของ Gantt Chart และระดับสถานะแสดงอย่างครบถ้วน
4.  กด **"บันทึก" (Save)** เพื่อดาวน์โหลดไฟล์ PDF รายงานทางการที่สวยงาม

## 📚 กรอบมาตรฐานอ้างอิงหลัก
นโยบายธรรมาภิบาลข้อมูลของ สทร. ในโครงการนี้ถูกออกแบบโดยผสานจุดเด่นของสองเสาหลักระดับประเทศ:
1.  **สำนักงานพัฒนารัฐบาลดิจิทัล (DGA):** ใช้เป็นกรอบการกำกับดูแลขั้นพื้นฐานด้านกฎหมายและระเบียบธรรมาภิบาลข้อมูลภาครัฐ
2.  **สำนักงานคณะกรรมการการรักษาความมั่นคงปลอดภัยไซเบอร์แห่งชาติ (สกมช.):** ใช้เป็นแนวปฏิบัติด้านความมั่นคงปลอดภัยของระบบข้อมูล สิทธิ์การจัดสรร และกระบวนการตลอดวงจรชีวิตข้อมูล
